# Marketing Campaign Template

## Purpose

The fillable campaign brief [CAMPAIGNS-AND-DEMAND-GENERATION.md](../06-marketing/CAMPAIGNS-AND-DEMAND-GENERATION.md) requires before launch — goal, audience, message, channel, and success metric in one place.

## When to Use

Before launching any coordinated campaign — required per [CAMPAIGNS-AND-DEMAND-GENERATION.md](../06-marketing/CAMPAIGNS-AND-DEMAND-GENERATION.md)'s "planning a campaign" section. Not needed for a single one-off piece of content with no coordinated push.

## Owner

Marketing, per [MARKETING-OPERATING-SYSTEM.md](../06-marketing/MARKETING-OPERATING-SYSTEM.md).

## Inputs

The ICP/segment being targeted ([IDEAL-CUSTOMER-PROFILE.md](../06-marketing/IDEAL-CUSTOMER-PROFILE.md)); the key message it carries ([POSITIONING-AND-MESSAGING.md](../06-marketing/POSITIONING-AND-MESSAGING.md)).

## Template Section

```markdown
# Campaign: <Name>

- **Owner:** <Owner>
- **Dates:** <Start> – <End>

## Goal
<The one primary goal — awareness, qualified leads, engagement.>

## Audience
<ICP/segment this targets.>

## Message
<Key message / proof point this campaign carries.>

## Channel(s)
<Wherever the audience actually is.>

## Success Metric
<The specific number that determines success — see MARKETING-METRICS.md.>

## Assets
- [ ] <asset — review status per MARKETING-REVIEW-AND-APPROVAL.md>

## Results (filled in after)
- **Outcome vs. goal:** <...>
- **What we'd change next time:** <...>
```

## Review Checklist

- [ ] One primary goal stated — not several competing ones.
- [ ] Every asset went through [MARKETING-REVIEW-AND-APPROVAL.md](../06-marketing/MARKETING-REVIEW-AND-APPROVAL.md) before launch.
- [ ] Tracking confirmed working before launch, not after.
- [ ] Results section filled in with a real hypothesis for next time, not just a number.
- [ ] No placeholder text remains.
